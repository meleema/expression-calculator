# Калькулятор математических выражений
Калькулятор математических выражений с поддержкой переменных, функций и скобок. Реализован на Java с использованием Maven, включает полный набор unit-тестов и документацию.

# Поддерживаемые операции
- Базовые арифметические операции: `+`, `-`, `*`, `/`
- Возведение в степень: `^`
- Скобки: `(`, `)` для изменения приоритета операций
- Приоритет операций: стандартный математический приоритет

# Математические функции
- Тригонометрические: `sin(x)`, `cos(x)`, `tan(x)`
- Логарифмические: `log(x)` (натуральный), `log10(x)` (десятичный)
- Алгебраические: `sqrt(x)`, `abs(x)`
- Сравнение: `max(a, b)`, `min(a, b)`
- Степенные: `pow(a, b)`

# Переменные
- Поддержка пользовательских переменных (любые буквенные идентификаторы)
- Интерактивный запрос значений переменных
- Регистронезависимые имена функций

# Технологии

- Java 11+
- Maven - управление зависимостями и сборка
- JUnit 5 - модульное тестирование
- JavaDoc - полная документация кода

# Установка и запуск
# Требования
- Java 11 или выше
- Maven 3.6 или выше

# Запуск тестов
mvn test

# Запуск приложения
mvn exec:java

# Сборка JAR
mvn package
java -jar target/expression-calculator-1.0.0.jar

# Принципы работы
1. Парсинг: `ExpressionParser` преобразует строку в последовательность токенов
2. Вычисление: `ExpressionEvaluator` обрабатывает токены с учетом приоритета операций
3. Функции: `FunctionRegistry` управляет математическими функциями
4. Ошибки: `ExpressionException` предоставляет детальную информацию об ошибках
